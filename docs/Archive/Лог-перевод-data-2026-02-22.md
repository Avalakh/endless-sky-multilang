# Краткий лог по переводу `language/ru/data`

## Цель
- Убрать оставшиеся английские строки в русских `.txt` данных и исправить автопереводчик, чтобы такие пропуски не повторялись.

## Что сделано
- Вручную поправлен `language/ru/data/human/frontier jobs.txt` (блок `Small Pirate gambling`) — заменены оставшиеся английские реплики.
- Проанализирован и доработан `tools/translate_data.py`:
  - исправлен сброс контекста диалогов на пустых/комментных строках (из-за этого пропускались части `conversation`/`choice`);
  - добавлен перевод standalone-строк в `"` внутри разговорного контекста;
  - добавлена защита кэша: если API вернул исходный английский текст, он не пишется в кэш как «готовый перевод»;
  - добавлен CLI-флаг `--skip-translated` для пропуска уже переведенных строк (эвристика: есть кириллица, нет латиницы).

## Команды, которые запускались
- Полный прогон переводчика:
  - `python tools/translate_data.py --skip-translated`

- Ошибок: `0`.
